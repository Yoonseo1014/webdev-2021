IntelliJ IDEA 단축키 15가지 알아보기

  1. Alt + enter: 오류 수정, 인텐션 액션, 검사(Import 되지 않은 Class import 시키기)
  2. F2: 다음 오류, 경고, 제안으로 점프
  3. Alt + 1: 프로젝트 창 열기
  4. Esc: 에디터로 포커스 이동, 변경 사항 적용하지 않고 팝업 창 닫기
  5. Ctrl + E: 최근에 열었던 파일 목록 보기
  6. Ctrl + B: 심볼의 선언으로 이동
  7. Alt + F7: 해당 항목이 사용된 위치 검색
  8. Ctrl + ctrl: 어떤 항목이든 실행
  9. Shift + Ctrl + W / Ctrl + W: 선택 영역 축소 / 선택 영역 확대
  10. Ctrl + / Comment Shift + Ctrl + / Comment: 주석 추가
  11. Shift + Ctrl + Enter: 현재 구문 자동 완성
  12. Ctrl + Alt + L: 자동 포매팅
  13. Shift + Ctrl + Alt + T: 사용 가능한 리팩토링 옵션 표시
  14. Shift + Ctrl + A: 단축키(액션) 검색
  15. Shift + Shift 전체 검색
